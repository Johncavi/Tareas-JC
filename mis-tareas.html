🔄 Sincronización en tiempo real

🟢
¡Sincronización activa!
Tus tareas se sincronizan en tiempo real entre todos tus dispositivos.
1

Crea tu proyecto en Firebase

Ve a console.firebase.google.com con tu cuenta Google → "Crear un proyecto" → nómbralo → continúa.

2

Activa Realtime Database

Menú izquierdo → "Realtime Database" → "Crear base de datos" → ubicación us-central1 → "Modo de prueba" → Habilitar.

3

Copia tu Database URL

Se ve así: https://tu-proyecto-default-rtdb.firebaseio.com/

4

Obtén tu API Key

⚙️ Configuración → General → "+ Agregar app" → ícono Web </> → Registrar → copia el valor de apiKey.
